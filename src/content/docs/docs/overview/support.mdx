Datum Cloud support is provided by our founding team via live chat and support@datum.net. You can also interact with us (as well as some awesome community members and open source contributors) on [Discord](https://link.datum.net/discord). Other resources include:
- [GitHub Discussions](https://github.com/orgs/datum-cloud/discussions) - Feature requests and changelog notices
- [Community Huddles](https://www.datum.net/community-huddle/) - Monthly calls hosted on Zoom.
