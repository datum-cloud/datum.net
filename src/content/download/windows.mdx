## Installation

<Tabs>
  <TabItem label="Homebrew">
  Install Datum via Homebrew with the following command:

  ```bash
  brew install datum-net/tap/galactic-vpc
  ```

  You can also download the latest version directly from our releases page.

## Configure and run

  1. Authenticate with your API token:
  ```bash
  galactic-vpc auth login --token <YOUR_API_TOKEN>
  ```

  2. Start a QUIC tunnel:
  ```bash
  galactic-vpc tunnel start --port 8080
  ```

  3. Your service is now reachable on the global edge.
